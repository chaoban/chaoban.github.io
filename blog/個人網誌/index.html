我建立這個個人網誌會用到的軟體或服務

By Chao Pang 22 July 2021

怕忘記步驟細節，在這裡做綱要。只凸顯其功能和程序，沒有安裝或設定細節。有需要的話Google一下即可。

下面這是會用到的軟體：

  • Github
  • Github Desktop
  • Hugo
  • Visual Studio Code
  • Typora
在Github上申請個人帳號
我的帳號為chaoban
那麼Github網址即為 https://github.com/chaoban
在Github上新增存放網誌的repo
建立一個repo名稱為chaoban.github.io
那麼個人網誌的位址即為 https://chaoban.github.io
在個人電腦上安裝Hugo，並下載網誌範本
我下載及使用 liva 這個範本
修改 liva網誌範本
可用VS Code編輯；用Typora來寫所見即所得的Markdown語法
產生網頁程式碼
啟動Hugo服務，下指令幫你產生網頁程式碼：
`hugo`
發佈到Github
配合Github Desktop使用
將生成的網頁程式碼(在資料夾 public 內)的所有檔案，放到chaoban.github.io內
push到Github上
Githun將你的網頁程式碼生成網頁
https://choaban.github.io
